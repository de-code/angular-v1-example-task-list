
Setup
=====

Install
-------
`npm install`

Test
----
`npm test`

Debug tests (single-run=false):

`npm test:debug`

And open the URL displayed, by default: http://localhost:9876/

Build
-----
`npm run build`

Start
-----
`npm start`

IDE / Visual Studio Code integration
------------------------------------
Build: Command + Shift + B

Test: Command + Shift + T
